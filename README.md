# python

## 가상환경 설정

1. 생성
```bash
python -m venv venv
```

2. 활성화
```bash
# linux / macOS
source venv/bin/activate
```

3. 비활성화
```bash
deactivate
```

4. jupyter lab 설치
```bash
pip install jupyterlab
```

5. jupyter lab 실행
```bash
jupyter lab
```
